# MoneyFlow

Conversor de moedas desenvolvido com HTML, CSS e JavaScript consumindo uma API de câmbio em tempo real.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Fetch API

## Funcionalidades

- Conversão de moedas
- Cotação em tempo real
- Histórico de conversões
- Inversão rápida de moedas
- Tratamento de erros
- LocalStorage
- Responsividade

## Conceitos Trabalhados

- Fetch API
- Async/Await
- Promises
- JSON
- Manipulação do DOM
- LocalStorage
- Tratamento de exceções

## Moedas Disponíveis

- BRL
- USD
- EUR
- GBP
- JPY
